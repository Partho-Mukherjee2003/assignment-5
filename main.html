Question-3: What is event delegation in the DOM?

Ans:Event delegation in the DOM lets a parent handle events for its child elements. Instead of adding many event listeners, you use one on the parent.

Question-4: What is event bubbling in the DOM?

Ans:Event bubbling in the DOM means when an event happens on a child element, it first runs on that element and then moves up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans: For create a element we can use .createElement().In this bracket we write the tag name that we want to create.
For add and remove elements we can use .appendChild() and .removeElement(). In this bracket we also write the element name that we want to add or remove. We must write element name using "" or '' in the bracket.
